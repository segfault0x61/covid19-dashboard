# COVID-19 Dashboard
This site is a COVID-19 dashboard which displays the worldwide and individual country rates of the COVID-19 vaccine by displaying deaths, recovered, and cases daily.

![Screenshot](https://raw.githubusercontent.com/segfault0x61/covid19-dashboard/master/res/1.png)


## Viewing The Site
Click [here](https://covid19-world-dashboard.netlify.app/) to visit the site.

## Technologies Used
* React
* Hooks
* Material UI
* Leaflet and React Leaflet
* Chart.js
